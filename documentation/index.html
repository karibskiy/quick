A) HTML Structure - top

To change countdown numbers, simply go to Index.html

<div class="counters cbp-so-side cbp-so-side-bottom" id="counter-1" data-countto="2">0</div>

change data-countto to numbers you need

To change Google Maps in contacts section, go to index.html

<iframe  src="https://maps.google.com/?ie=UTF8&amp;ll=52.24462,14.067993&a
mp;spn=0.491896,1.352692&amp;t=m&amp;z=10&amp;output=embed"></iframe>

and change link to the one you have in google maps SEND options. You can use coodrinatates or direct adress link. Depends what adress yo have

to change video from top play button, go to index.html

<a class="voverlay" href="http://www.youtube.com/v/qb1xc1RjrxQ?autoplay=1
&amp;rel=0&amp;enablejsapi=1&amp;playerapiid=ytplayer">

and again change ID to the one you need.

to change data for loan calculator:

<input class="input-amount" type="text" value="" data-maxval="400" 
data-valstep="50" data-round="10">
 data-maxval="400">

Change data-maxval to the one you actually needed.

B) CSS Files and Structure - top

I'm using thre important CSS files in this theme. The first one (index.css) contains all basic css styles.Rest Css is for responsive version and IE compability. Probably CSS should not be edited.

To change background go to:

.wraper.bg-raw {

If you would like to edit a specific section of the site, simply find the appropriate label in the CSS file, and then scroll down until you find the appropriate style that needs to be edited.

C) JavaScript - top

This theme imports two Javascript files.

  1. jQuery
  2. jQuery UI
  3. jQuery.counters.js
  4. SlidesJS
  5. MapsJS
  6. niceScroll
  7. site specific scripts
  1. jQuery is a Javascript library that greatly reduces the amount of code that you must write.
  2. jQuery UI enhances previous library with additional effects.
  3. jQuery.counters.js
  4. SlideJS is used to create slider effect for images. It is called in site specific scripts file (js/site.js), like this:
TO Change video in Video version background go to init.js 
 $(function() {
 $('body').tubular({videoId: '9d8wWcJLnFI'}); 
 });

In to videoId put your Video Youtube ID

D) PSD Files - top

I've included one PSD file with this folders inside:

  1. Footer - basic copyright and subscribe form
  2. Navigation - menu and logo
  3. Services - what company offer shortly
  4. About - short intro text about company
  5. Customers - some client feedback
  6. Main slider - basic loan calculator
  7. Contacts - loan contact field

If you'd like to change the main background, open "set backgrounds", make the necessary adjustments, and then save the file. Do the same for the subscribe filed & header text.
