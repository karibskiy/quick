jQuery Bubbler.js plugin

jQuery Bubbler.js is a plugin for jQuery to apply a floating bubble effect on DOM elements

Usage

1. Copy the files jquery.bubbler.js to your project. Paste the following code right before the closing tag in your HTML document. Make sure these lines are below the line where you include jQuery. Change the red parts to reflect your situation.

      <script src="path/to/jquery.bubbler.js"></script>

2. The plugin will work on any DOM element. Use $(".ClassName") to apply it to an element with a class name or $("#elementID") with and ID. Or just create a new one like:

      <div id="bubbles"></div>

3. Apply css style to the element for width and height:

      #bubbles{
        width:200px;
        height:200px;
      }

4. Apply the following code to start the bubbler plugin:


      $("#bubbles").bubbler({
        color: "#adadad",  //color of the bubbles. Shading will be automatically 
        ammount: 20,  // bubble count
        min: .1,      // Minimal size of the bubble in percentage (greater than zero, less than 1)
        max: .4,      // Maximal size of the bubble in percentage
        time: 10,     // Bubble move time (greater is slower)
        vertical:true,  // Move the bubbles vertically
        horizontal:true  // Move the bubbles horizontally
      });
